persistencia_io.py 
import json

# Escribir texto plano con Context Manager
with open("archivo.txt", "w") as f:
    f.write("Hola mundo\nSegunda línea\n")

# Serializar estructuras nativas a JSON
datos = {"nombre": "Juan", "edad": 25, "activo": True}
with open("datos.json", "w") as f:
    json.dump(datos, f, indent=2)

💡
La ventaja de los Context Managers: Utilizar la sentencia with garantiza la gestión automática de recursos en el sistema operativo. No importa si ocurre una excepción inesperada a mitad del proceso, Python se asegurará de cerrar correctamente el puntero del archivo en memoria (stream descriptor), evitando fugas de memoria o corrupción de datos.

Modos de Apertura de Buffers y Serialización de Estructuras

Controlar el comportamiento de apertura determina cómo interactúa el intérprete con el espacio físico del disco. Además, la combinación de estos flujos con formatos ligeros como JSON facilita la interoperabilidad del software:

Catálogo de Modos Básicos:
• "r" : Modo Lectura. Posiciona el puntero al inicio; lanza un error si el archivo no existe.
• "w" : Modo Escritura. Trunca (vierte y vacía) el archivo por completo antes de escribir; si no existe, lo crea.
• "a" : Modo Append. Añade la información al final del archivo existente sin alterar los registros previos.
• "x" : Modo Creación Exclusiva. Construye un nuevo archivo; aborta y lanza un error si la ruta ya está ocupada.
procesamiento_json.py 
import json

# Recuperación de diccionarios desde un archivo JSON
with open("datos.json", "r") as f:
    cargado = json.load(f)
    print(cargado["nombre"])  # Salida: Juan

# Guardado masivo de arreglos de objetos estructurados
usuarios = [
    {"nombre": "Ana", "edad": 20},
    {"nombre": "Luis", "edad": 30}
]
with open("usuarios.json", "w") as f:
    json.dump(usuarios, f)

Ejercicio 2: Serialización e hidratación estática de propiedades (JSON)

Persiste un mapa de coordenadas espaciales traduciéndolo a formato JSON y deserialízalo hacia una variable mapeada nativa.

serializa_coordenadas.py 
import json

datos = {"x": 10, "y": 20}
with open("datos.json", "w") as f:
    json.dump(datos, f)

with open("datos.json", "r") as f:
    cargado = json.load(f)
    print(cargado)  # Salida: {'x': 10, 'y': 20}

Ejercicio 3: Hidratación e iteración cíclica de arreglos compuestos

Carga una matriz compleja de registros de usuario codificados en JSON y recorre recursivamente sus propiedades para extraer atributos particulares.

iteracion_usuarios.py 
import json

usuarios = [
    {"nombre": "Ana", "edad": 20},
    {"nombre": "Luis", "edad": 30}
]

with open("usuarios.json", "w") as f:
    json.dump(usuarios, f)

with open("usuarios.json", "r") as f:
    data = json.load(f)
    for u in data:
        print(u["nombre"])

# Salida:
# Ana
# Luis
