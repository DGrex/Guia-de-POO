Unpacking en
Python

Domina la extracción y distribución atómica de datos coleccionables. Descubre cómo desempaquetar de forma directa secuencias, mapeos y estructuras iterables asignándolas a variables independientes en una sola línea de código.

unpacking_basico.py 
# Unpacking posicional básico
a, b = [1, 2]
print(a, b)  # Salida: 1 2

# Capturas avanzadas usando el operador *
primero, *resto, ultimo = [1, 2, 3, 4, 5]
print(primero)  # Salida: 1
print(resto)    # Salida: [2, 3, 4]
print(ultimo)   # Salida: 5
a, b = lista *args **kwargs Sintaxis
💡
¿Qué es el Unpacking? Es una técnica expresiva nativa de Python que elimina la necesidad de acceder a colecciones de forma manual mediante índices repetitivos (como hacer a = lista[0]). Permite mapear instantáneamente estructuras iterables enteras en variables unitarias bien nombradas, mejorando drásticamente la legibilidad del código.

Desempaquetado en Firmas de Función e Iteraciones Cíclicas

La versatilidad del unpacking permite esparcir parámetros posicionales hacia argumentos de funciones, fusionar diccionarios en caliente o deconstruir estructuras compuestas directamente dentro de sentencias for.

funciones_y_mapeos.py 
def suma(a, b, c):
    return a + b + c

numeros = [1, 2, 3]
print(suma(*numeros))  # Esparcido posicional. Salida: 6

dict1 = {"a": 1}
dict2 = {"b": 2}
combinado = {**dict1, **dict2}  # Fusión de diccionarios
print(combinado)  # Salida: {'a': 1, 'b': 2}
bucles_desempaquetados.py 
coordenadas = [(1, 2), (3, 4), (5, 6)]

for x, y in coordenadas:
    print(f"x={x}, y={y}")

# Salida:
# x=1, y=2
# x=3, y=4
# x=5, y=6

Ejercicio 2: Argumentación dinámica por expansión (Splat Operator)

Pasa una lista completa de datos numéricos discretos hacia los parámetros requeridos por una función de multiplicación sin indexar de manera individual.

funcion_splat.py 
def multiplicar(a, b, c):
    return a * b * c

lista = [2, 3, 4]
print(multiplicar(*lista))  # Salida: 24
