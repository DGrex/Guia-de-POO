El Constructor
__init__
en Python

El método __init__ es un método especial en Python que se ejecuta automáticamente cuando se crea una nueva instancia de una clase. Su función principal es inicializar los atributos del objeto.

Ir a Ejercicios ⬅ Volver al inicio
persona.py 
# Definimos una clase Persona
class Persona:
    def __init__(self, nombre, edad):
        # El constructor recibe parámetros y los asigna a atributos del objeto
        self.nombre = nombre
        self.edad = edad

# Creamos un objeto de la clase Persona
p1 = Persona("Ana", 25)

# Mostramos los atributos inicializados
print(p1.nombre)  # Salida: Ana
print(p1.edad)    # Salida: 25
💡
Concepto Clave: Piensa en self como una referencia al propio objeto que se está creando en ese instante.

Ejercicio 1: Clase Coche con atributos marca y modelo

Crea una clase coche que reciba e inicialice de forma correcta sus propiedades iniciales.

coche.py 
class Coche:
    def __init__(self, marca, modelo):
        # Guardamos los valores recibidos en atributos
        self.marca = marca
        self.modelo = modelo

# Instanciamos un objeto de tipo Coche
c1 = Coche("Toyota", "Corolla")

# Mostramos los atributos inicializados
print(c1.marca)   # Salida: Toyota
print(c1.modelo)  # Salida: Corolla

Ejercicio 2: ¿Qué sucede si no defines __init__?

Si no defines __init__, Python crea un constructor por defecto de forma implícita que no inicializa ningún atributo. Podrás instanciar y crear objetos de la clase sin problemas, pero estos no tendrán valores iniciales asignados automáticamente a menos que los definas de manera externa más adelante.
