💡
Casting necesario: La función input() siempre devuelve una cadena de texto (str). Por lo tanto, es obligatorio usar funciones como int() o float() para convertir los datos si requieres operar matemáticamente con ellos.

Entrada de Datos

Uso de input() para capturar variables dinámicas desde la terminal del usuario.

entrada_datos.py 
nombre = input("Ingrese su nombre: ")
edad = int(input("Ingrese su edad: "))   # casting a int
altura = float(input("Ingrese su altura: ")) # casting a float

print(f"{nombre} tiene {edad} años y mide {altura} m")
#️⃣
⚙️

Casting (Conversión de Tipos)

Consiste en transformar de manera explícita el tipo de dato de una variable en otro diferente. Las herramientas fundamentales nativas integradas para realizar esta acción sobre flujos de entrada de texto son:

• int(): Convierte texto a números enteros.
• float(): Convierte texto a números decimales.
• str(): Transforma cualquier objeto en su representación de texto.
