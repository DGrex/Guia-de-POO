# Ejemplo básico de condicional
edad = 18

if edad >= 18:
    print("Eres mayor de edad")   # Se ejecuta porque edad es 18
else:
    print("Eres menor de edad")

Concepto clave: Las estructuras condicionales permiten desviar y ramificar el camino de ejecución del código de un programa basándose en si la evaluación de una condición booleana resulta en True o False.
#️⃣
⚙️

¿Cómo funcionan if, elif y else?

Python evalúa el código de arriba hacia abajo de manera secuencial:

• if: Es el punto de partida obligatorio. Si la condición es verdadera, ejecuta su bloque de código y omite el resto de la estructura.
• elif: (Abreviación de else if) Permite encadenar múltiples evaluaciones alternativas opcionales si las condiciones previas resultaron falsas.
• else: Bloque final opcional que captura cualquier caso sobrante que no haya cumplido ninguna de las condiciones anteriores.

Ejercicio 1: Clasificación de números (Signo)

Crea un script con ramificaciones para determinar si un número dado es positivo, negativo o idénticamente cero.

clasifica_numero.py 
numero = -3

if numero > 0:
    print("El número es positivo")
elif numero < 0:
    print("El número es negativo")
else:
    print("El número es cero")

# Salida: El número es negativo
