Variables y
Tipos de Datos
en Python

Una variable es un espacio en memoria asignado para almacenar un valor. En Python no necesitas declarar el tipo explícitamente; este se determina y asigna de forma dinámica según el valor guardado.

Ir a Ejercicios ⬅ Volver al inicio
variables.py 
# Asignamos diferentes tipos de valores a variables
numero_entero = 10       # int
numero_decimal = 3.14    # float
texto = "Hola Denis"     # str
es_valido = True         # bool

# Mostramos los valores y sus tipos
print(numero_entero, type(numero_entero))   # Salida: 10 
print(numero_decimal, type(numero_decimal)) # Salida: 3.14 
print(texto, type(texto))                   # Salida: Hola Denis 
print(es_valido, type(es_valido))           # Salida: True
int float str bool
💡
Tipos Básicos: int para enteros (10, -5); float para decimales (3.14, -0.5); str para texto ("Hola"); y bool para valores lógicos (True/False).

Ejercicio 3: Diferencia entre int y float

int representa números enteros exactos y sin parte fraccionaria, mientras que float se utiliza para representar números reales con precisión decimal. Por ejemplo, 10 es procesado como un tipo int, mientras que 10.0 corresponde a un float.
