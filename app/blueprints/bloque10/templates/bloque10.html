diccionario_basico.py 
# Definimos un diccionario de estudiante
estudiante = {
    "nombre": "Ana",
    "edad": 22,
    "carrera": "Ingeniería"
}

# Accedemos a un valor usando la clave
print(estudiante["nombre"])   # Salida: Ana

# Agregamos una nueva clave-valor
estudiante["promedio"] = 9.1
print(estudiante)  # Salida: {'nombre': 'Ana', 'edad': 22, ...}

Concepto de Asociación: A diferencia de las listas y tuplas que recuperan elementos a través de su posición numérica indexada, los diccionarios utilizan un concepto de tabla Hash. Esto permite acceder a cualquier valor de forma directa e instantánea conociendo su palabra clave única.

Reglas de Estructura e Inmutabilidad

Para manipular de manera óptima diccionarios en Python, debes recordar las siguientes restricciones técnicas:

• Claves Únicas: No se permiten duplicados para una misma clave. Declarar una clave existente sobreescribirá el valor anterior sin lanzar errores.
• Claves Inmutables: Las claves deben ser tipos de datos inmutables (strings, números, tuplas). No puedes usar una lista como clave de un diccionario.
• Valores Dinámicos: Al contrario de las claves, los valores asignados pueden ser cualquier tipo de objeto (listas, funciones, o incluso otros diccionarios anidados).

Ejercicio 2: Iteración cíclica en paralelo (.items)

Desempaqueta las llaves y valores de un diccionario de forma simultánea aplicando bucles iterativos avanzados.

itera_diccionario.py 
for clave, valor in pais.items():
    print(clave, ":", valor)

# Salida:
# nombre : Ecuador
# capital : Quito
# poblacion : 18000000
