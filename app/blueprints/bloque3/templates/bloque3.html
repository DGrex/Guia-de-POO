Aritméticos Comparación Lógicos Identidad (is)
💡
Diferencia clave: El operador == compara exclusivamente el contenido de dos variables, mientras que el operador is evalúa si apuntan a la misma referencia en memoria.

Operadores de Comparación

Evalúan relaciones de igualdad o magnitud, devolviendo siempre un valor booleano.

comparacion.py 
x = 10
y = 20

print(x == y)  # Igualdad → False
print(x != y)  # Diferente → True
print(x > y)   # Mayor → False
print(x < y)   # Menor → True
print(x >= 10) # Mayor o igual → True
print(y <= 20) # Menor o igual → True

Operadores Lógicos

Permiten combinar o interceptar múltiples condiciones lógicas consecutivas.

logicos.py 
cond1 = True
cond2 = False

print(cond1 and cond2) # AND → False
print(cond1 or cond2)  # OR → True
print(not cond1)       # NOT → False

Contenido vs Memoria (== vs is)

Demostración práctica del comportamiento de asignación por valor y referencia.

identidad.py 
a = [1, 2]
b = [1, 2]
c = a

print(a == b) # True → mismo contenido
print(a is b) # False → objetos distintos
print(a is c) # True → misma referencia

Precedencia de Operadores

Orden jerárquico estricto en el que Python interpreta una expresión matemática.

precedencia.py 
x = 2 + 1 * 2 % 2 + (2 ** 1) // 2
print(x)  # Resultado: 3

# Orden jerárquico: ** → % → // → * → +

Ejercicio 3: Desglose de Precedencia

Al evaluar la expresión 2 + 1 * 2 % 2 + (2 ** 1) // 2 obtendremos un resultado final de 3. Esto se debe a que Python primero procesará el contenido de los paréntesis y potencias, seguido por los operadores de multiplicación, residuo y división entera de izquierda a derecha, efectuando las sumas al último escalón de la ejecución.
