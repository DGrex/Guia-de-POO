Concepto de Set: Un conjunto es una estructura de datos indexada internamente mediante tablas hash que no permite elementos repetidos. Además, a diferencia de las listas o tuplas, los conjuntos no tienen un orden garantizado, por lo que no es posible acceder a sus posiciones usando índices numéricos como [0].

Operaciones de Álgebra de Conjuntos y Métodos Nativos

Python implementa operadores bit a bit para simular operaciones matemáticas de conjuntos de manera óptima, además de métodos integrados para alterar su mutabilidad de forma segura.

operaciones_algebraicas.py 
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}

print(A | B)  # Unión → {1, 2, 3, 4, 5, 6}
print(A & B)  # Intersección → {3, 4}
print(A - B)  # Diferencia → {1, 2}
print(A ^ B)  # Diferencia simétrica → {1, 2, 5, 6}
metodos_conjuntos.py 
conjunto = {1, 2, 3}
conjunto.add(4)       # Agregar elemento
conjunto.remove(2)    # Eliminar (lanza KeyError si no existe)
conjunto.discard(5)   # Eliminar de forma segura (no lanza error si no existe)

print(conjunto)       # {1, 3, 4}
