¿Qué es una lista? Es una estructura de datos ordenada y mutable que permite agrupar múltiples objetos en una sola variable. A diferencia de otros lenguajes, en Python las listas pueden contener de manera simultánea elementos de diferentes tipos de datos.
#️⃣
⚙️

Características Fundamentales

Para sacar el máximo provecho de las colecciones indexadas, recuerda tres pilares clave:

• Ordenadas: Mantienen estrictamente la secuencia original en la que fueron guardados sus ítems, basándose en índices numéricos desde el cero.
• Mutables: Su contenido puede alterarse de forma libre en tiempo de ejecución (añadir, remover o reemplazar elementos) sin alterar su identidad original.
• Indexación Negativa: Python permite acceder desde el extremo final hacia atrás de forma nativa utilizando números negativos (donde -1 representa la última posición).

Ejercicio 1: Manipulación de ítems (Insertar y Eliminar)

Utiliza métodos nativos integrados para alterar dinámicamente el tamaño y el contenido interno de un arreglo de cadenas.

modifica_lista.py 
frutas = ["manzana", "banana", "naranja"]

# Agregar un elemento al final
frutas.append("uva")
print(frutas)  # Salida: ['manzana', 'banana', 'naranja', 'uva']

# Eliminar un elemento específico
frutas.remove("banana")
print(frutas)  # Salida: ['manzana', 'naranja', 'uva']

Recorre ordenadamente cada casilla de la lista extrayendo sus valores singulares mediante una estructura de control iterativa.

Ejercicio 3: Segmentación avanzada mediante Slicing

Aplica delimitadores de rangos de índices (sub-listas) para seccionar subconjuntos iniciales y terminales de un arreglo.

slicing_listas.py 
numeros = [10, 20, 30, 40, 50]

# Obtener los primeros 3 elementos
print(numeros[:3])  # Salida: [10, 20, 30]

# Obtener los últimos 2 elementos
print(numeros[-2:])  # Salida: [40, 50]
