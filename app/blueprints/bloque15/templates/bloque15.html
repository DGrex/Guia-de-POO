# Una función de orden superior recibe o retorna otra función
numeros = [1, 2, 3, 4, 5, 6]

# Combinación elegante: Filtrar pares y elevarlos al cuadrado
resultado = list(
    map(lambda x: x ** 2,
        filter(lambda x: x % 2 == 0, numeros))
)
print(resultado)  # Salida: [4, 16, 36]

¿Qué son las HOF? Las *Higher-Order Functions* (HOF) tratan a las funciones como ciudadanos de primera clase. Permiten abstraer flujos repetitivos de datos pasándoles pequeños fragmentos de código lógico a través de expresiones lambda (funciones anónimas), reduciendo drásticamente las mutaciones de estado y los bucles manuales.

Tríada Fundamental de la Programación Funcional

Python implementa de forma nativa tres funciones de orden superior clave para transformar, restringir y consolidar colecciones iterables de datos.

herramientas_funcionales.py 
from functools import reduce

numeros = [1, 2, 3, 4]

# 1. MAP: Aplica una función de transformación a cada casillero
cuadrados = list(map(lambda x: x ** 2, numeros))
print(cuadrados)  # Salida: [1, 4, 9, 16]

# 2. FILTER: Conserva únicamente los ítems que validan un predicado booleano
pares = list(filter(lambda x: x % 2 == 0, numeros))
print(pares)      # Salida: [2, 4]

# 3. REDUCE: Acumula y procesa la colección secuencialmente hasta un único valor
suma = reduce(lambda x, y: x + y, numeros)
print(suma)       # Salida: 10

Ejercicio 3: Consolidación y agregación consecutiva (reduce)
