contador = 1

while contador <= 5:
    print("Iteración", contador)
    contador += 1

💡
¿Qué son los bucles? Las estructuras cíclicas o bucles permiten ejecutar un mismo bloque de instrucciones múltiples veces. Son herramientas cruciales para evitar la duplicación de código y procesar colecciones de información.

El Bucle for y Secuencias

Uso de for para examinar y recorrer de manera automática los elementos de una secuencia (listas, cadenas de texto, rangos numéricos, etc.).

bucle_for.py 
for numero in range(1, 6):
    print("Número:", numero)

# Salida:
# Número: 1
# Número: 2
# Número: 3
# Número: 4
# Número: 5

Ejercicio 1: Suma acumulativa (1 al 10) con while

Desarrolla un algoritmo que sume los valores de un rango numérico de manera incremental mientras se cumpla la condición.

suma_acumulada.py 
suma = 0
i = 1

while i <= 10:
    suma += i
    i += 1

print("La suma es:", suma)  # Salida: La suma es: 55

Ejercicio 2: Recorrido secuencial de listas

Itera directamente a través de un arreglo iterable de strings aplicando formateo dinámico en cada elemento.

recorrido_lista.py 
nombres = ["Ana", "Luis", "María"]

for nombre in nombres:
    print("Hola", nombre)

# Salida:
# Hola Ana
# Hola Luis
# Hola María

Ejercicio 3: Filtrado condicional interno (Pares)

Combina un bucle con bifurcaciones lógicas aritméticas para discriminar e imprimir exclusivamente números pares.

numeros_pares.py 
for numero in range(1, 11):
    if numero % 2 == 0:
        print(numero)

# Salida:
# 2
# 4
# 6
# 8
# 10
